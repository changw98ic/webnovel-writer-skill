# @changw98ic/core

核心类型和 Skill 定义 - 通用网文创作框架。

## 安装

```bash
npm install @changw98ic/core
```

## 使用

```typescript
import { Skill, SkillContext, SkillResult } from '@changw98ic/core';
import { ProjectState, Entity, EntityType } from '@changw98ic/core/types';
```
