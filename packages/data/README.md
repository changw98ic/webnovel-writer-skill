# @changw98ic/data

数据层 - 状态管理、实体索引、RAG 检索。

## 安装

```bash
npm install @changw98ic/data
```

## 使用

```typescript

import { IndexManager } from '@changw98ic/data';
import { RAGAdapter } from '@changw98ic/data';
```
